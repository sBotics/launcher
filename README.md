<h2 id="english"> <img src=".github/README/en.png" width="16"> &nbsp;English </h2> 

### Compiling instructions
Download dependencies: 
```bash
yarn install
```

Run application:
```bash
yarn start
```

Baixando dependências: 
```bash

```

Rodar a aplicação:
